# (C# 코딩) LoginScreen

##개요
-C# 프로그래밍 학습
-1줄 소개: 아이디와 패스워드를 입력하면 로그인 성공으로 처리하는 프로그램
-사용한 플랫폼: 
 -C#, .NET Windows Forms, Visual Studio, GitHub
-사용한 컨트롤: Login, txtID, txtpw, LogB

-사용한 기술과 구현한 기능:
 - Visual Studio를 이용하여 UI 디자인
 - enter, leave를 사용하여 textbox 기능 추가
 - MessageBoxIcon.Error 경고 메시지 띄우기
 - Enter키 이벤트 처리
